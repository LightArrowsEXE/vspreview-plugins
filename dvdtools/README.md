# DVD Tools

A collection of tools for working with DVDs.

## ISO Browser

A plugin for browsing DVDs.
Supports everything `vssource.IsoFile` supports.

### Features

- Load ISO files
- Navigate through titles
- View title properties
- Dump audio tracks to WAV/AC3 files
